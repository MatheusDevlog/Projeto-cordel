# 📜 Projeto Cordel

Este é um projeto desenvolvido com foco em prática de **HTML e CSS**, inspirado na estética e na narrativa da literatura de cordel. A proposta foi criar uma página visualmente agradável, utilizando efeitos de rolagem, tipografia estilizada e imagens de fundo que ajudam a contar a história de forma envolvente.

## 🚀 Sobre o projeto

O site apresenta um poema em formato de cordel, organizado em seções que alternam entre fundos fixos e conteúdo textual, criando um efeito visual interessante durante a navegação.

Durante o desenvolvimento, foram trabalhados conceitos como:

* Estruturação semântica com HTML
* Estilização com CSS
* Efeito *parallax* simples
* Responsividade básica
* Uso de fontes personalizadas

## 🎯 Objetivo

O principal objetivo deste projeto foi reforçar habilidades no front-end, especialmente na criação de layouts modernos e experiências visuais utilizando apenas HTML e CSS, sem o uso de JavaScript.
